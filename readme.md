# 114-1 資料庫管理 期末專案

本倉庫為 114-1「資料庫管理」課程之期末專案。內容包含專題提案（LaTeX 與已編譯 PDF），以及之後將逐步加入的程式碼與資料庫設計。

## 專案簡介（短版）

- 專題目標：以關聯式資料庫為核心，設計並實作一個解決實際需求的應用（主題將於開發過程中明確化）。
- 主要成果：
  - 概念與邏輯資料模型（ERD/Schema）
  - 基本與進階查詢（含索引與效能考量）
  - 簡易 API 或操作介面（視實作範圍而定）
